# Projet de Graphe de Réseau en Python

Ce projet permet de modéliser un réseau informatique sous forme de graphe, en utilisant la bibliothèque `networkx` pour calculer les chemins les plus rapides entre différents nœuds (ordinateurs, routeurs, serveurs) et `matplotlib` pour visualiser le graphe.

---

## Fonctionnalités

- **Modélisation d'un réseau** : Représentation des ordinateurs, routeurs et serveurs sous forme de nœuds.
- **Calcul des chemins les plus rapides** : Utilisation de l'algorithme de Dijkstra pour trouver les chemins les plus rapides entre les nœuds en fonction de la latence.
- **Visualisation du graphe** : Affichage du graphe avec les nœuds et les arêtes, incluant les latences.

---

- Python 3.x
- `networkx`
- `matplotlib`
